# LifeSync AI

> 나만의 디지털 생활 관리 비서 + AI 커뮤니티

---

## 프로젝트 개요

**LifeSync AI**는 캘린더, 할 일, 일기장을 하나로 통합하고, AI가 사용자의 일상을 분석하여 인사이트를 제공하는 앱입니다.

### 핵심 가치
1. **통합된 삶의 기록**: 일정, 할 일, 감정을 한 곳에서
2. **AI 자문단**: 3명의 AI 페르소나가 나를 분석하고 응원
3. **자연어 인터페이스**: 말하듯이 기록하고 관리

### 영감의 출발점
- **멀티북(Multi-book)**: 여러 AI가 내 데이터를 보고 토론하는 개념
- **Generative Agents (Stanford/Google)**: "몸통이 감시하다가 뇌를 깨우는" 아키텍처

## 핵심 기능

### 💬 AI 채팅
자연어로 일정, 할 일, 일기를 기록합니다.

### 🤖 AI 커뮤니티
3명의 AI 페르소나가 사용자 행동에 자동으로 반응합니다.

| 페르소나 | 역할 |
|---------|------|
| 🧮 아리아 | 데이터 기반 인사이트 |
| 💛 모모 | 따뜻한 축하와 위로 |
| 🎯 세이지 | 다음 액션 제안 |

### 📅 캘린더
시간의 흐름을 조망하는 뷰어

### ✅ 할 일
칸반 스타일의 깔끔한 목록 관리

### 📝 일기장
감정과 함께 하루를 기록

### 🔍 검색/기록
통합 검색 + 활동 이력 추적

### ☁️ 클라우드 동기화
모든 데이터 실시간 백업 및 멀티 디바이스 연동
