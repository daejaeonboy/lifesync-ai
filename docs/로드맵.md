# 로드맵

> LifeSync AI의 향후 개선 방향

---

## 기술적 고도화 (Engineering)

### 1. Cloud Sync & Auth
- 현재 `localStorage` 기반 데이터를 Supabase/Firebase로 마이그레이션
- PC와 모바일 간 데이터 연동
- 사용자 인증 시스템

### 2. Real AI Integration
- 시뮬레이션 응답 → 실제 LLM (GPT, Gemini) 연동
- 사용자 데이터를 Vector DB(RAG)에 임베딩
- "나보다 나를 더 잘 아는" 초개인화된 회고 제공

예시:
> "지민님, 지난달에는 월요일마다 기분이 안 좋으셨네요. 내일 일정 조정을 추천드려요."

---

## 기능 확장 (Feature Expansion)

### 1. Data Visualization
- 감정 + 성취도 데이터 결합
- **월간 라이프 밸런스 차트** 제공
- 동기부여 효과

### 2. Voice Input
- 모바일 환경 고려
- 음성으로 일정 등록, 하루 회고

### 3. Smart Notifications
- 맥락 인식 능동적 알림
- "오후 3시 회의 전에 점심 시간 확보하세요"

---

## UX 디테일

### 1. Dark Mode
- 차분한 Off-White 테마와 대비되는 'Midnight Blue' 톤
- 야간 사용성 향상

### 2. Routine Templates
- 매일 반복되는 할 일(루틴) 템플릿화
- 클릭 한 번으로 불러오기

### 3. 템플릿 커스터마이징
- 사용자 생성 템플릿 저장
- 검색 결과에서 바로 수정/완료 처리

---

## 우선순위 제안

### Phase 1: 신뢰성
1. 액션 확인/Undo + 변경 로그 ✅ (완료)
2. 데이터 내보내기/삭제 ✅ (완료)

### Phase 2: 편의성
1. 알림 제어: Nudge 설정/끄기 + Quiet Hours
2. 통합 검색 ✅ (완료)

### Phase 3: 지능화
1. LLM 연동
2. 감정 분석 기반 동적 체인
3. 대화 요약 및 인사이트 추출

## 리스크 관리

| 리스크 | 영향 | 대응 |
|-------|------|------|
| 의도 오해로 잘못된 일정/할 일 생성 | 신뢰 저하 | 실행 전 확인, Undo |
| 능동 알림 과다 | 피로/이탈 | 빈도·시간대 설정 |
| 감정 대화 안전성 | 과도한 의존 | 경계 문구, 전문가 연결 안내 |
| 데이터 프라이버시 | 법적 리스크 | 암호화, 삭제/내보내기 |
