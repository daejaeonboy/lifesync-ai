# 디자인 철학

> LifeSync AI의 핵심 디자인 컨셉: **"Digital Sanctuary (디지털 서재)"**

---

## 핵심 철학

**'복잡한 현대인의 일상 속, 가장 정돈된 나만의 공간'**

수많은 생산성 도구들이 기능 과잉으로 오히려 피로감을 주는 반면,
LifeSync AI는 사용자가 **기록하는 행위 자체에 몰입하고 위로받을 수 있는 UX**를 지향합니다.

> "좋은 디자인은 가능한 한 디자인을 하지 않는 것이다." - Dieter Rams

---

## 시각적 언어 (Visual Language)

### 색상 시스템: **흑백 기반 (Monochrome)**
**현재 브랜드 컬러가 정해지지 않았으므로, 모든 UI는 흑백/그레이스케일로 구성합니다.**

| 용도 | 색상 | 설명 |
|------|------|------|
| 배경 | `#F7F7F5` / `#FBFBFB` | 따뜻한 Off-White |
| 주 텍스트 | `#37352f` | 깊이감 있는 Warm Gray |
| 보조 텍스트 | `#9b9a97` / `#787774` | 정보 위계 표현 |
| 보더 | `#E9E9E8` / `#d3d1cb` | 섬세한 1px 구분선 |
| 강조 | `#37352f` (검정) | 중요 요소에만 사용 |

### 🚫 사용하지 않는 것
- **컬러풀한 색상**: 파랑, 빨강, 초록, 노랑 등
- **그라데이션**: 모든 그라데이션 배경/효과
- **화려한 애니메이션**: 정적이고 차분한 느낌 유지

### ✅ 향후 브랜드 컬러
브랜드 컬러가 정해지면 아래 요소에 **포인트 색상**으로 적용:
- 주요 CTA 버튼
- 완료 체크박스
- 네비게이션 활성 상태

---

## 디자인 원칙

1. **Off-White & Warm Gray**: 차가운 '도구'가 아닌 따뜻한 '종이'의 질감
2. **Minimal Borders**: 강한 그림자 대신 얇고 섬세한 1px 보더
3. **Typography Focus**: 장식적인 아이콘을 줄이고, 폰트 굵기와 투명도로 정보 중요도 표현
4. **Frictionless**: 사용자의 사고 흐름을 끊지 않는 인터랙션
5. **Static & Calm**: 정적이고 차분한 인터페이스, 애니메이션 최소화
6. **No Gradients**: 단색 배경 사용, 그라데이션 금지

---

## 모듈별 설계 의도

### 📅 캘린더: "흐름을 보다 (Flow)"
- **Timeline Sidebar**: 일정을 '리스트'가 아닌 '시간의 연속'으로
- **Frictionless Entry**: 빈 공간 더블 클릭으로 일정 추가
- **Property Modal**: Notion 스타일의 정돈된 속성창
- **흑백 태그**: 중요도는 진하기로 구분 (검정 = 중요)

### 💬 AI 채팅: "말로 하다 (Speak & Do)"
- **Intent-Based Parsing**: 자연어로 의도 파악
- **Action Feedback**: 무엇이 어디에 추가되었는지 명확한 피드백
- **Suggestion Chips**: 처음 사용자도 쉽게 시작

### ✅ 할 일: "단순함의 힘 (Simplicity)"
- **칸반 보드**: 깔끔한 컬럼 레이아웃
- **원형 체크박스**: 미니멀한 완료 표시 (흑백)
- **인라인 입력**: 클릭 한 번으로 즉시 입력
- **목록 이름 편집**: 클릭하여 목록 이름 수정 가능

### 📝 일기장: "집중과 기록 (Focus)"
- **Emotional Dots**: 감정은 이모지로만 표현
- **Micro-Interaction**: 최소한의 부드러운 애니메이션

### 🤖 AI 커뮤니티: "나를 읽다 (Read Me)"
- **Feed-Style Layout**: AI 분석을 잡지처럼 편안하게
- **체인형 대화**: AI들이 순차적으로 인사이트 제공
- **흑백 아바타**: 컬러 대신 이모지와 회색 배경

---

## 덜어내기의 미학

모든 UI 요소에서 '이것이 정말 필요한가?'를 끊임없이 질문했습니다.

- 체크박스를 단순한 원형으로 변경
- 감정 선택을 아이콘 3개로 단순화
- 캘린더의 격자 선을 아주 연하게 처리
- **모든 컬러 제거** → 흑백/그레이스케일만 사용
- **그라데이션 제거** → 단색 배경만 사용

모든 결정은 **사용자의 시선을 본질적인 콘텐츠(텍스트)로 유도**하기 위함입니다.
